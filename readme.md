## üî• Featured: 100xDevs Cohort 3.0

Harkirat Singh's **Cohort 3.0** is a rigorous Full Stack & Web3 bootcamp. This repository will systematically document the entire curriculum.

### üåê Track 1: Full-Stack Web Development & DevOps
- **Foundation:** HTML, CSS, JavaScript, VS Code, Terminal mastery.
- **MERN Stack:**
  - **Frontend:** React.js, State Management (Recoil/Redux), Tailwind CSS.
  - **Backend:** Node.js, Express.js, JWT Auth, Zod Validation.
  - **Database:** MongoDB, PostgresQL, Prisma ORM.
- **Advanced Web:**
  - Next.js 14 (App Router), Server Actions.
  - TypeScript for type-safe applications.
  - Monorepos (Turborepo) & Serverless (Cloudflare Workers).
  - Real-time Communication: WebSockets, WebRTC.
- **DevOps:**
  - Docker, Containerization.
  - CI/CD Pipelines (GitHub Actions).
  - Kubernetes (K8s), AWS (EC2, S3, ECS).
  - Monitoring: Prometheus, Grafana.

### ‚õìÔ∏è Track 2: Web3 & Blockchain (Solana & Ethereum)

- **Ethereum ecosystem:**
  - Solidity smart contracts.
  - Hardhat/Foundry for testing & deployment.
  - DApps with Ethers.js / Wagmi.
- **Solana Ecosystem:**
  - Rust programming language.
  - Solana Program Library (SPL).
